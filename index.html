#实验一 在Github上用Markdown创建技术博客

###一、实验目的
1.熟悉Github网站
2.掌握Markdown语法

###二、实验任务
1.注册Github账号
2.创建Gihub Pages repository
3.发布第一篇博客
4.学习Markdown
5.利用Hexo生成博客并部署到Github

###三、实验步骤与结果
1.注册github账号：Yanghan888

2.创建名为 Yanghan888.githuo.io 的 Gihub Pages repository

3.发布第一篇博客

4.在 Markdown 上进行博客编辑

5.  本地配置Hexo，具体步骤如下：
   *安装git
   *安装node
   *设置镜像为淘宝
       cnpm install -g cnpm --registry=https://registry.npm.taobao.org
   *安装hexo
       cnpm install -g hexo-cli
   *初始化站点
       hexo init hexo
       cd hexo
       Npm install
   *浏览本地站点
      hexo sever
   6.  在github上部署hexo，具体步骤如下：

*注册github账号：Yanghan888

*创建名为 Yanghan888.githuo.io 的 Gihub Pages repository

*发布第一篇博客

在 Markdown 上进行博客编辑 修改配置文件_config.yml的deploy部分
type:git
Repository:https://github.com/Yanghan888/Yanghan.github.io.git
Branch:master
配置site部分
title:Yanghan888的博客
Author:Yanghan888
Language:zh-CN 安装hexo-deployer-git
npm install hexo-deployer-git –save
配置git全局邮箱与名字
git config –global user.email”1023688472@qq.com”
git config –global user.name”Yanghan888” 生成博客
hexo generate
*部署博客
hexo deploy

  1. 添加博客文章
    hexo new myblog
    *用VS修改文章
    hexo\source_posts\myblog.md
  2. 生成并发布到github
    hexo d
    hexo g

###四、实验小结
实验过程中遇到了很多问题，走了很多弯路，很多次不成功又从头来过，后来经过自己查阅资料，多次探索才能初步得到博客的原始模型，今后还将不断完善，直至博客界面新颖美观，有自己的个性。
